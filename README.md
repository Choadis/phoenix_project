# PhoenixDex

A pokedex made in Phoenix

## Getting Started

* [Rails](https://rubyonrails.org/)
* Ruby

### Prerequisites

* Be sure to make sure ruby is installed on your computer with:
```ruby -v```

### Installing

* Install rails with `gem install rails`
